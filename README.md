# AgroSmart IoT Irrigation

Application mobile Expo React Native pour la gestion intelligente de l'arrosage agricole.

## Prérequis

- Node.js
- npm
- Expo CLI via `npx`

## Installation

```bash
cd frontend
npm install
```

## Lancer le projet

```bash
npx expo start
```

Pour lancer en web :

```bash
npx expo start --web
```

## Structure

```text
frontend/
  src/
    components/
    data/
    navigation/
    screens/
    theme/
```

Le backend n'est pas encore connecté. Les données IoT sont simulées pour préparer plus tard l'intégration ESP32, Firebase ou API.
